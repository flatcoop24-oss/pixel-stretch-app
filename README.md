# Pixel Stretch PWA

아이폰에서 Safari로 열고 **공유 → 홈 화면에 추가**를 선택하면 앱처럼 설치됩니다.

## 주요 기능
- 직선형 픽셀 스트레치
- 브러시형 픽셀 스트레치
- 가로/세로 방향
- 길이/두께/강도 조절
- 실행 취소/원본 복원
- PNG 저장 및 iOS 공유 시트
- 서비스워커 기반 오프라인 실행

## GitHub Pages
저장소의 **Settings → Pages → Build and deployment → Source**를 `GitHub Actions`로 지정하면 자동 배포됩니다.

배포 주소: `https://flatcoop24-oss.github.io/pixel-stretch-app/`
